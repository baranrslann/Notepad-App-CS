# Notepad App

C# ve Windows Forms kullanılarak geliştirilmiş, tamamen işlevsel bir Windows Not Defteri (Notepad) uygulaması. Bu proje; temel masaüstü uygulama mantığını, özel klavye kısayollarını (hotkeys) ve dosya sistemi yönetimini göstermektedir.

### ✨ Özellikler ve Kısayollar
* **Dosya İşlemleri:** Yeni Pencere (`Ctrl+Shift+N`), Aç (`Ctrl+O`), Kaydet (`Ctrl+S`), Farklı Kaydet (`Ctrl+Shift+S`) ve Yazdır (`Ctrl+P`).
* **Görünüm Ayarları:** Yakınlaştır (`Ctrl + '+'`), Uzaklaştır (`Ctrl + '-'`) ve Varsayılan Yakınlaştırma (`Ctrl + 0`).
* **Metin Biçimlendirme:** Metni Kaydır (Word Wrap) ve Özel Yazı Tipi/Boyutu Seçimi (Font).
* **Ekstralar:** Anlık Saat/Tarih Ekleme (`F5`) ve Durum Çubuğu (Status Bar) gizleme/gösterme.

### 🛠️ Kullanılan Teknolojiler
* **Dil:** C#
* **Framework:** .NET / Windows Forms
* **Geliştirme Ortamı:** Visual Studio 2022

### ✨ Features & Shortcuts
* **File Operations:** New Window (`Ctrl+Shift+N`), Open (`Ctrl+O`), Save (`Ctrl+S`), Save As (`Ctrl+Shift+S`), and Print (`Ctrl+P`).
* **View Customization:** Zoom In (`Ctrl + '+'`), Zoom Out (`Ctrl + '-'`), and Restore Default Zoom (`Ctrl + 0`).
* **Text Formatting:** Word Wrap toggle and Custom Font/Style selection.
* **Extras:** Quick Time/Date insertion (`F5`) and Status Bar visibility control.

### 🛠️ Technologies Used
* **Language:** C#
* **Framework:** .NET / Windows Forms
* **IDE:** Visual Studio 2022
